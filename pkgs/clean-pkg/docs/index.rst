Genie Clean
===========

Scared of rommon mode and always want your testbed running in sane condition?

Here comes ``genieclean`` which provides modular sections to bring up your device
to whatever desired state.

This section provides an overview on how to use clean stages during a ``Genie`` run.

We recommend reading through these sections in order, to help understand all
concepts associated with clean methodologies within ``Genie``.
